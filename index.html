Focus on your business

Deploying applications and DevOps works is a pain that all the development teams wants to forget. Docker containers definitely help us to do that on developers' laptop but not yet for the whole application lifecycle, that often needs a dedicated team to manage servers and deployment. With Continuous-Pipe, you can deploy your applications with the standard docker-compose.yml file and with a very little configuration on the Docker cluster you wants (Kubernetes or Google Container Engine for now).

We help you through the whole application lifecycle

Feature environments

We all agree that the development is easier when we use feature branches. By deploying these feature branches into feature environments, we have an isolated environment that can be used by QA and the final customer to improve the feedback loop.

GitHub integration

You only have to activate your GitHub repository from the UI. The status of the deployments will be displayed on GitHub via its statuses and the addresses of your feature environments commented on the pull-requests.

Continuous deployment

Once you've done with your feature branch and everybody is happy, just merge the pull-request and CP will automatically updates the environment associated with the principal branch like it is doing with feature branches.
